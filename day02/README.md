# Day 02 - Scientific Calculation (Dilution)

## Program Description

This program calculates dilution using the formula:

C1 * V1 = C2 * V2

The user enters:
- Stock concentration (C1)
- Stock volume (V1)
- Desired concentration (C2)

The program calculates:
- Final volume (V2)

---

## Sample Input

C1 = 10  
V1 = 5  
C2 = 2  

## Output

V2 = 25.0

---

## How to run

```bash
python3 calculation.py
