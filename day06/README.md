# Day 06 - Protein Analysis with UniProt API

## Description

This program retrieves protein information from the UniProt database using its REST API.

The program analyzes protein sequence properties such as:
- amino acid frequencies
- protein length
- hydrophobic amino acid percentage

## About the Database

UniProt is a scientific database that provides comprehensive information about proteins, including:
- protein sequences
- biological functions
- organism information
- molecular annotations

It is widely used in bioinformatics and molecular biology research.

## Requirements

Install dependencies with:

```bash
pip install -r requirements.txt
```

## Running the Program

```bash
python protein_analysis.py
```
